# 鸿蒙实训

## 项目简介

本项目是吉林大学企业实训课程的作业，包含两个基于 HarmonyOS ArkUI 框架开发的应用，用于学习和实践鸿蒙应用开发技术。

## 注意事项

- DevEco Studio 使用的版本为 6.0.1.260，虚拟机使用的是 HarmonyOS 6.0.1(21)
- 版本更新较快，实际使用时建议以官方最新版本为准

## 技术栈

- **框架**：HarmonyOS ArkUI
- **开发语言**：ArkTS (TypeScript-based)
- **API 版本**：HarmonyOS Next
- **构建工具**：Hvigor

## 项目结构

```text
.
├── data-viz-数据可视化/          # 数据可视化期末大作业
├── jilinuniversity-homework/    # 吉林大学作业项目 - ArkUI 组件学习演示
├── tkbrush-app/                 # TKBrush 刷题应用 - 完整功能应用
└── 课程文档/                     # 课程相关文档（实训总结模板等）
```

## 功能特性

### jilinuniversity-homework（组件学习演示）

用于学习和演示 ArkUI 各种组件的项目，涵盖以下模块：

- **布局组件**：弹性布局、列布局、行布局、堆叠布局、滚动组件
- **列表与网格**：列表组件、网格组件、轮播图
- **容器组件**：标签页组件
- **基础组件**：边框样式、边距内边距、通用组件
- **数据与渲染**：数据传递、条件渲染、循环渲染
- **高级功能**：路由测试、构建器测试、方法调用
- **数据存储**：SQLite、首选项存储

### tkbrush-app（刷题应用）

功能完整的刷题和社区交流应用：

- **首页**：刷题功能、试卷列表、考试功能、网页浏览
- **消息**：消息通知、消息管理
- **社区**：社区首页、文章详情、发布文章
- **我的**：用户信息、个人设置、关于我们、修改密码、修改昵称
- **登录注册**：用户登录、用户注册、忘记密码

## 运行说明

### jilinuniversity-homework

1. 使用 DevEco Studio 打开 `jilinuniversity-homework` 目录
2. 连接设备或启动模拟器
3. 点击运行按钮编译并安装应用

### tkbrush-app

1. 使用 DevEco Studio 打开 `tkbrush-app/tkbrush-app` 目录
2. 连接设备或启动模拟器
3. 点击运行按钮编译并安装应用
