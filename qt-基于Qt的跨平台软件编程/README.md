# EasyWord

## 项目简介

本项目是吉林大学基于 Qt 的跨平台软件编程课程的作业，基于 Qt 框架开发的简化版文字处理软件。

## 注意事项
- 无课程文档，我直接把要求整理成题目.md。

## 技术栈

- **语言**：Python 3.x
- **GUI 框架**：PyQt5

## 项目结构

```text
.
├── EasyWord.py    # 主程序
├── 功能介绍.md     # 软件功能介绍
├── 题目.md         # 课程要求整理
└── README.md      # 项目说明文档
```

## 功能特性

- 富文本编辑（中英文混排）
- 字体样式：粗体、斜体、下划线、颜色、高亮
- 段落对齐：左对齐、居中、右对齐、两端对齐
- 文件操作：新建、打开、保存（HTML/TXT）
- 编辑功能：撤销、重做、剪切、复制、粘贴
- 快捷键支持
- 状态栏：行号、列号、字数统计

## 运行说明

```bash
python EasyWord.py
```
