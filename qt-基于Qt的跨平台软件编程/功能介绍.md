# EasyWord 软件功能介绍

## 一、软件概述

**EasyWord** 是一款基于 Qt 框架开发的简化版文字处理软件，参照 WPS 和 Microsoft Word 进行设计。本软件实现了文字处理中最核心、最常用的功能，采用跨平台的 Qt 技术，可在 Windows、macOS、Linux 等操作系统上运行。

## 二、实现的功能清单

### 1. 文字输入功能
- **富文本编辑**：支持在编辑区自由输入文字，支持中英文混排
- **自动换行**：编辑区宽度自适应，文字自动换行
- **光标定位**：通过鼠标或键盘方向键精确定位光标位置

### 2. 基本格式设定

| 功能 | 说明 | 操作方式 |
|------|------|----------|
| **字体设置** | 支持系统中所有已安装字体（宋体、黑体、微软雅黑等） | 工具栏字体下拉框 |
| **字号设置** | 支持 6~72 号字，自由调节文字大小 | 工具栏字号选择框 |
| **加粗** | 将选中的文字设为粗体 | 工具栏 B 按钮 / Ctrl+B |
| **斜体** | 将选中的文字设为斜体 | 工具栏 I 按钮 / Ctrl+I |
| **下划线** | 为选中的文字添加下划线 | 工具栏 U 按钮 / Ctrl+U |
| **字体颜色** | 修改选中文本的前景色 | 工具栏 "A" 按钮 |
| **高亮颜色** | 为选中文本添加背景色高亮 | 工具栏 "高亮" 按钮 |
| **左对齐** | 段落靠左对齐 | 工具栏 "左对齐" 按钮 |
| **居中对齐** | 段落居中对齐 | 工具栏 "居中" 按钮 |
| **右对齐** | 段落靠右对齐 | 工具栏 "右对齐" 按钮 |
| **两端对齐** | 段落两端对齐 | 工具栏 "两端对齐" 按钮 |

### 3. 文字编辑功能

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| **撤销** | 撤销上一步操作 | Ctrl+Z |
| **重做** | 恢复被撤销的操作 | Ctrl+Y |
| **剪切** | 剪切选中的内容到剪贴板 | Ctrl+X |
| **复制** | 复制选中的内容到剪贴板 | Ctrl+C |
| **粘贴** | 将剪贴板内容粘贴到当前位置 | Ctrl+V |
| **全选** | 选中编辑区全部内容 | Ctrl+A |

### 4. 文件操作功能

| 功能 | 说明 |
|------|------|
| **新建** | 创建一个新文档，支持未保存提示 |
| **打开** | 打开已有的 HTML 或纯文本文件 |
| **保存** | 将当前文档保存到文件（支持 HTML 和 TXT 格式） |
| **另存为** | 将文档以新文件名或新格式保存 |
| **退出** | 关闭程序，修改未保存时自动提示 |

### 5. 辅助功能
- **状态栏信息**：实时显示当前光标所在的行号、列号
- **字数统计**：实时显示文档中的字符数量（不含空格和换行）
- **修改标记**：文档被修改后标题栏显示 "*" 标记，提醒用户保存
- **关闭确认**：关闭未保存文档时弹出确认对话框

## 三、界面设计

软件采用经典的 **QMainWindow** 布局：
- **顶部**：菜单栏（文件、编辑、格式、帮助）+ 工具栏（常用快捷按钮）
- **中央**：富文本编辑区，占据主要窗口空间
- **底部**：状态栏，显示行列位置和字数统计

## 四、技术特点

1. **跨平台**：基于 Qt 框架，一次编写可在多个平台编译运行
2. **富文本支持**：使用 QTextEdit 控件，天然支持 HTML 富文本格式
3. **信号与槽机制**：采用 Qt 的信号/槽实现界面交互，代码解耦清晰
4. **快捷键支持**：所有常用操作均支持标准键盘快捷键
