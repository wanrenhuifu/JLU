# 智能医疗问诊记录分析平台

## 项目简介

吉林大学软件体系结构课程结课大作业。系统通过AI技术自动分析医患对话，提取关键医疗信息，生成结构化电子病历。

**核心功能**：语音转文本 | 症状提取 | 术语标准化 | 合规检查
