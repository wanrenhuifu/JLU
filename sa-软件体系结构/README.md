# 智能医疗问诊记录分析平台

## 项目简介

本项目是吉林大学软件体系结构课程结课大作业。系统通过 AI 技术自动分析医患对话，提取关键医疗信息，生成结构化电子病历。

## 注意事项

- 文档图表使用 Mermaid 语法渲染
- 课程文档中已去掉学生分组信息表，防止泄露隐私
- 让 AI 设计方案中涉及使用大模型时，注意使用最新版本的模型，避免使用过时的模型
- 本项目为架构设计文档，不包含代码实现。

## 技术栈

- **文档编辑**：Markdown
- **图表绘制**：Mermaid
- **版本控制**：Git

## 项目结构

```text
.
├── 智能医疗问诊记录分析平台-架构设计文档.md    # 详细架构设计文档
├── 题目.md                                      # 项目需求整理
├── prompt.txt                                   # 引导大模型生成架构设计文档的提示词
└── 课程文档/                                    # 课程相关参考文档
```

## 功能特性

- 语音转文本
- 症状提取
- 术语标准化
- 合规检查
